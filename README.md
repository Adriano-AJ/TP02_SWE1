# 📦 Gerenciador de Produtos

Sistema Web desenvolvido para a disciplina de SWE1 (2026).

## 👥 Desenvolvedores
* **Adriano Júnior de Souza Almeida**
* **Arthur Lanzilotti Farjanes**

## 🛠️ Tecnologias
* **Linguagem:** Java

## 🚀 Funcionalidades
* **Cadastro:** Registro de novos produtos.
* **Listagem:** Visualização de todos os itens em tabela.
* **Edição:** Alteração de informações existentes.
* **Exclusão:** Remoção de itens do sistema.

**Nota:** Projeto focado no uso de Servlets e manipulação de objetos em memória.
